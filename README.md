# Proyecto Final - Tienda Mística

Estimado profesor, este proyecto corresponde al trabajo final del curso.
Sitio web de venta de productos místicos (velas, inciensos, piedras) desarrollado con **React**, **Vite** y **Firebase**.

## Descripción

Este proyecto es un e-commerce que permite:
- Visualizar un catálogo de productos por categoría.
- Agregar productos al carrito de compras.
- Completar el checkout usando un formulario con integración a Firebase.

## Tecnologías

- React 18
- Vite
- Firebase
- Bootstrap 5
- React Router DOM
- React Icons
