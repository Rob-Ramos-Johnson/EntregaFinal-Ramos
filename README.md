# ProyectoFinal+Ramos - Tienda Mística

E-commerce SPA en React + Firebase (lista para conectar).

## Contenido
- Productos: inciensos, velas, piedras energéticas (mock data).
- React + Vite, React Router, Context API para carrito.
- Bootstrap + custom CSS.
- Firestore integration (ready to connect via env variables).

## Instrucciones

1. Instalar dependencias:
```bash
npm install
```

2. Crear archivo `.env` en la raíz con estas variables (reemplaza con tus credenciales de Firebase):
```
VITE_FIREBASE_API_KEY=...
VITE_FIREBASE_AUTH_DOMAIN=...
VITE_FIREBASE_PROJECT_ID=...
VITE_FIREBASE_STORAGE_BUCKET=...
VITE_FIREBASE_MESSAGING_SENDER_ID=...
VITE_FIREBASE_APP_ID=...
```

3. Correr en desarrollo:
```bash
npm run dev
```

4. Si querés subir a GitHub, crea repo público llamado `ProyectoFinal+Ramos` y subí este contenido.
